Project: Exploration and analysis of anime data with Python

Project's Description This project I have made with the IDE
Spyder and the public repository of datasets called: Kaggle

Source of Information: 
animes : https://www.kaggle.com/datasets/marlesson/myanimelist-dataset-animes-profiles-reviews
animes_2:

This tools are ones(til now) that I most used and I feel more
comfortable during the elaboration of personal projects or
college projects

In this project there would be some methods that do not adjust
to the dataset "animes", the main reason, is that I started to
work with "animes_2"but, in the middle way, the table has many
data in blank, so was difficult to work. At the end, I change 
the dataset and mantain the methods because I think it would be
useful in other projects.

Content:
* cargar_data = Functions to extract data.
* conseguir_data = Functions to get data from the dataset.
* graficar_data = Functions to graphic the data got of the dataset.
* limpiar_data = Functions that "clean" the table in order to
make easy the work of analyze data.
* modificar_data = Functions that change the structure of
the table.
